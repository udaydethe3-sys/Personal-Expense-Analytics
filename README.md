## What This Project Contains

- Category wise breakdown (Food, Transport, Rent, Entertainment, Shopping)
- Pie chart showing annual expense distribution
- Summary dashboard with key insights

## Key Findings
- Rent is the largest fixed monthly expense
- Shopping and Entertainment spike during festive months
- Total annual expense tracked and analyzed
